# MASKED RUNNER - Global Game Jame 2026

This game is the submission for the **Global Game Jam 2026** event from the team **METTelhead**.

- Location: Hamburg (InnoGames)
- Theme: MASK

## About the Game

**MASKED RUNNER** is a fast-paced runner game built with React Three Fiber where you must navigate through obstacles while avoiding enemies that can damage you. Collect power-ups like healing potions, speed boosts, and vaccination immunity to survive as long as possible!

## Tech Stack

- **React** + **TypeScript**
- **React Three Fiber** (@react-three/fiber) - 3D rendering in React
- **Tailwind CSS** - Styling
- **Vite** - Build tool
- **Zustand** - State management
- **Lucide React** - Icons

## Local Development

Install dependencies:

```bash
pnpm i
```

Run the development server:

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to see the game in the browser.
